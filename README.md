# vite-react-ts

First contact using:

- [Vite.js](https://vitejs.dev/)

- [TypeScript](https://www.typescriptlang.org/)

- [React](https://reactjs.org/)

- [PostCSS](https://postcss.org/)

- [TailwindCSS](https://tailwindcss.com/)

## Commands

### Creation

```
npm create vite@latest vite-react-ts -- --template react-ts
```

### Development

```
npm run dev
```

### Production Build

```
npm run build
```

#### Preview

```
npm run preview
```
